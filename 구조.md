reward

tactical16
= 기본 눈, 기본 입력 16개

student/my_observation.py
= 네가 눈을 직접 새로 설계하는 파일

student/my_reward.py
= 네가 채점 기준을 직접 설계하는 파일

policy model
= observation을 받아 action을 내는 학습된 블랙박스

observation = AI가 보는 것
reward = AI가 잘했는지 채점하는 것
policy/model = 그걸 보고 조종 명령을 내는 블랙박스
action = 조종 명령
